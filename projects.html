2024
Wolno-Nam: Trauma UX Design Web Rework
Summary

This course combines the study of user experience (UX) design with an immersive travel experience in Poland, focusing on how design can be informed by and respond to trauma, especially historical war related trauma. Through visits to significant sites, interactions with local communities, and hands-on design project for a refugee center, students will learn to create UX solutions that are sensitive to traumatic experiences and promote healing and resilience.

Experience Gained
  • Understanding of Trauma-Informed Design: Gained insights into how UX design can be informed by and respond to trauma, particularly historical war-related trauma.
  • Cultural Sensitivity and Inclusion: Learned the importance of incorporating cultural sensitivity and inclusion in UX solutions to support communities affected by trauma.
  • Hands-On UX Design for Social Impact: Created a set of wireframes for the Wolno-Nam Refugee Foundation, focusing on creating solutions that promote healing and resilience.
  • Empathy in Design Processes: Enhanced understanding of empathy in design, ensuring that the user experience considers the emotional and psychological state of users.
  • Site-Specific UX Solutions: Adapted UX solutions to local needs by creating both English and Polish versions of the wireframes to cater to location differences.
  • Practical Design Constraints: Applied practical knowledge in design by avoiding bright colors, sharp angles, and unnecessary elements that could negatively impact trauma survivors.
  • Community Engagement: Engaged with local communities through visits to significant sites, gaining firsthand understanding of the context and challenges faced by the community.

2022
Roomster - UX Mobile Application Design
Summary

In this course, my team and I explored the iterative UX design process to create a roommate-finding application called Roomster. We learned to develop user personas and utilized design tools like Figma and Whimsical to create wireframes. We focused on various design fidelities and conducted extensive user research to ensure our application met user needs effectively. Throughout the project, we applied Nielsen Norman Group best practices to refine our design approach. Additionally, we emphasized accessibility by adhering to WCAG contrast standards and carefully selected font faces and color schemes using Paletton. This project provided a practical, hands-on experience in crafting a user-centered design solution.

Experience Gained
  • Iterative UX Design Process: Learned the steps of creating effective user personas and engaging in rapid ideation with 'Crazy 8's' exercises.
  • Wireframing and Prototyping: Developed skills in wireframing and utilized tools like Figma and Whimsical for high-quality prototyping.
  • User Research: Conducted extensive user research to inform design decisions and improve user interface usability.
  • Agile Environment: Practiced UX design within an agile framework, enhancing collaborative and iterative design capabilities.
  • Accessibility and Compliance: Applied WCAG contrast standards and other accessibility guidelines to ensure designs are user-friendly for all.
  • Visual Design Skills: Focused on selecting appropriate font faces and crafting color schemes using tools like Paletton.
  • Industry Best Practices: Integrated Nielsen Norman Group’s best practices into design work to enhance user experience quality.

Competition Eagle - Java Advanced Programming

Utilizing Java, we focused on building these games to understand and apply class hierarchies effectively, which helped in organizing our code and enhancing the scalability of our application. We also employed Java Swing to create the graphical user interface, which provided a visually engaging and interactive experience for users. Working collaboratively in a team of four, we learned to integrate our individual contributions into a cohesive application, demonstrating the practical application of advanced programming concepts and teamwork in software development.

Experience Gained

  • Class Hierarchies: Learned about and implemented class hierarchies to structure our code logically and efficiently.
  • Java Swing Usage: Employed Java Swing to develop the graphical user interface, enhancing the visual appeal and user interaction of the games.
  • Team Collaboration: Worked closely with teammates to integrate various components into a seamless application, fostering strong teamwork skills.
